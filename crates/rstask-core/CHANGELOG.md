# rstask-core

## 0.3.0 — 2026-02-18

### Minor changes

- [59036bd](https://github.com/bruits/rstask/commit/59036bd730d2862038e17740726f74af8dc4537c) Rework CLI and convert yaml to Markdown.
  
  `rstask` now uses Markdown files instead of YAML for tasks. This allows you to use and edit tasks notes as actual Markdown instead of cramping it inside the `notes` property like in `dstask`. Additionally, `rstask show <id>` will nicely render the task's notes to the terminal. — Thanks @Princesseuh!
